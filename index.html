Mise en place de l'IDE

On va utiliser le logiciel eclipse fournit par le SDK android ICI (32 ou 64 bits).

Après avoir téléchargé cela, nous avons eclipse et le SDK manager. Il vaut mieux décompresser le .zip dans un endroit bien définit car nous y ferons référence dans eclipse.

Une fois décompressé, lancer le "SDK Manager.exe" et se laisser guider. Cela ne concerne pas eclipse mais la version android + emulateur android (ADT plugin). Cet outil permet de gérer l'installation des modules android comme la version du SDK, les APIs des versions android. On va prendre le Android SDK TOols + l'API 10 (v2.3.3 d'android). Selectionner les package puis cliquer sur installer. Cocher les licenses pour pouvoir installer.

Eclipse : Le dossier contenant eclipse est à mettre par exemple dans:

-Pour le 32 bits : C:ProgramFiles/Eclipse

-Pour le 64 bits : C:Programmes/Eclipse

On va lancer Eclipse maintenant juste pour créer le dossier "workspace" des projets. Il peut être fermé ensuite.

Fin de l'étape. Maintenant passons à GIT !

Inscription sur GitHub

Pour commencer il faut un compte sur www.github.com. Cela créera un pseudo sous lequel chaque commit sera enregistré. On prendra l'exemple de "jhonr" comme pseudo dans la suite.

Rejoindre Polycraft

Après l'inscription il va falloir rejoindre l'organisation Polycraft qui a en charge le projet SlippyJuice est stocké.

Générer un clef SSH (permet d'authentifier le PC de développement)

Pour participer au développement il va falloir générer une clef SSH sur l'ordinateur de développement. Pour cela une section et dédiée sur le site github ICI.

Avec le logiciel pour windows, la clef ssh est créée sans le voir. Donc on télécharge l'application GitHub. Installation de cette dernière puis lancement. La phase de détection des repositories peut être passée.

Après s'être connecté sur l'application à l'aide du pseudo et du mot de passe on va importer le projet. On va le mettre directement dans le dossier "workspace" d'éclipse. Pour cela on va dans "Polycraft" puis sur le projet SlippyJuice clic droit et "clone to". On selectionne alors le dossier "workspace". Puis on valide.

Le projet est cloné dans le dossier voulu, la clef SSH est créée dans les documents dans /.ssh et enfin cette clef est automatiquement ajoutée au compte github. Magnifique, on peut fermer l'application GitHub.

On va repasser à la phase finale dans Eclipse.

Plugin Eclipse pour GIT

Pour se faciliter le développement on va installer un plugin eclipse pour gérer les commits. Il s'appelle "Eclipse EGIT".

Pour l'installer on va démarrer l'Eclipse téléchargé avec le SDK d'Android. Une fois dans Eclipse, aller dans Help/Install New Software. Dans le champ "Work with" mettre l'adresse du plugin : http://download.eclipse.org/egit/updates puis cliquer sur "Add" et donner un nom quelconque à ce lien. Une fois validé,

Installer l'application gitHub.
