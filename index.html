Mise en place de l'IDE

On va utiliser le logiciel eclipse fournit par le SDK android ICI (32 ou 64 bits).

Après avoir téléchargé cela, nous avons eclipse et le Android SDK Tools qui permet d'installer les version android que l'on veut. Pour le moment choisir la version par défaut car tout cela peut être fait plus tard. La procédure à suivre ensuite est indiqué sur cette page : ICI.

Lancer le .exe donc et se laisser guider. Cela ne concerne pas eclipse mais la version android + emulateur android (ADT plugin).

Le dossier contenant eclipse est à mettre là où on le souhaite. Mais les chemins conseillé sont les suivants: -Pour le 32 bits : C:ProgramFiles/Eclipse -Pour le 64 bits : C:Programmes/Eclipse

Fin de l'étape. Maintenant passons à GIT !
